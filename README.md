# Gesture Recognition

A real-time hand gesture recognition system built using Python, OpenCV, and Machine Learning.

## Features

* Detects hand gestures using webcam
* Real-time recognition
* Computer vision based interaction

## Technologies Used

* Python
* OpenCV
* MediaPipe
* Machine Learning

## How to Run

1. Install Python
2. Install required libraries
3. Run the Python file

```bash
python main.py
```
